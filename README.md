# SlotFinder — Smart Car Parking System

An AI-powered car parking solution that predicts parking slot availability 
in real time using time and location data.  
Built as part of a research project with a full published paper.

---

## 🧠 What Does It Do?

Finding a parking spot in a busy city wastes time, fuel, and money.  
**SlotFinder** solves this by using machine learning to:

- 📍 Predict **which parking slots will be free** based on location and time of day
- 📊 Analyse **patterns in parking behaviour** to improve accuracy
- ⏱️ Help drivers **plan ahead** instead of driving around searching

---

## 📌 Key Highlights

- ✅ Built a **multivariate machine learning model** to forecast parking availability
- ✅ Used **spatio-temporal data** (location + time combined) for smarter predictions
- ✅ Conducted full research — from data collection to model evaluation
- ✅ Published a research paper on the findings
